# 第五讲 推荐系统 前言

# 1、前言

在前四讲中我们已经开发完新闻推荐系统的基本功能，比如搜索，随机推荐，新闻查看等，这一讲我们将会专注于推荐功能的开发，同时，我们的技术也将从原本的 java web 升级到 springboot，数据库操作升级到 mybatis 框架，前端使用 vue2 的脚手架进行开发，因此，以上知识欠缺或者不熟练的童鞋要抓紧补课了，对于基础较好的同学，相信跟着我的步伐，你会很快实现这个系统~

# 2、开发环境

暂无变化，总的见第一讲，当然为了减少学习成本，我会尽量少用不必要的技术，虽然使用它们可能会更加方便，但我想让本教程的门槛更低一点~

# 3、开发模式

本项目我还是基于前后端分离的开发模式，前端后端分开讲解，在这里先开发后端，把接口都准备好，然后再开发前端，当然在实际中是可以同时开发的，只要把接口统一好即可，不过在这里我们是单人开发，所以就按顺序开发即可
