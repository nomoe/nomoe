# 第二讲 前端页面开发-vue高阶版-脚手架版

# 1、前言

我们已经开发了 vue2 的基础版，现在来开发高阶版，也就是用脚手架的方法，适合基础比较好的童鞋

# 2、环境搭建

# 1、新建项目

vue 创建脚手架项目的方式有很多，我们选择手动创建，打开 webstorm ，选择空项目，然后输入项目名，点击创建，如果是用 vscode则直接新建文件夹，然后右键->通过code打开即可

# 2、创建 vue 项目

打开终端，输入 vue create .，然后回车

" />" />

然后根据图示操作即可

Babel 和 Router 是必选的，然后其他的根据自己兴趣来，使用上下键移动，使用空格选择或取消选择，然后选择完成后回车就行

版本我们选 2.x 就行

是否选择路由历史模式，我们选择否，路径丑一点没关系

选择插件配置的位置，我们选择放在 package.json ，因为我们本次不改动任何插件配置，所以没必要，让目录简洁亿点不香嘛~

是否存储此方案作为预设方案，我选 no ，你们随意，个人觉得手动配置繁琐是繁琐，但是比较稳

稍等一会儿就创建好了

然后在终端输入 yarn serve 启动服务器，当然如果你用的是 npm 的话，运行 npm run serve 也行，不过我推荐使用 yarn

出现这个界面就说明启动成功，打开 Chrome，输入 http://localhost:8080/ ，出现该界面则说明OK了

当然，由于我装了 vue 的插件，所以右上角的小图标就点亮了~
