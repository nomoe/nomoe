# 一、开发首页静态页面

# 依赖简介

本次用到的依赖： normalize.css 、jQuery.js，没有用 bootstrap，尽量不引入太多库，因为项目结构比较简单，直接手撕就行，感兴趣的同学自行学习、升级项目

normalize.css：是一个可以定制的CSS文件，它让不同的浏览器在渲染网页元素的时候形式更统一

jquery 和 normalize.css 的下载链接，推荐 bootcdn 或者 staticfile.org： https://cdn.bootcdn.net/ajax/libs/jquery/3.6.0/jquery.js (opens new window) https://cdn.bootcdn.net/ajax/libs/normalize/8.0.1/normalize.css (opens new window)
