# 2、实现首页与后端交互

# 1、测试前后端通信

首先我们要引入一个库，用于发送 ajax 请求，叫做 axios，在 github 上足足有 86k 颗星，那为什么不引入jquery，有的人会说杰哥 jquery 我比较熟悉，但是在 vue 中引入 jquery 并不是一个合适的选择，第一，jquery 是直接操作 dom，而 vue 则不提倡直接操作 dom；第二，jquery 这个库非常大，就为了发个请求而引入这么大的库，确实有点不合理，所以我推荐使用 axios，vue 官方也推荐使用

安装：

# npm
npm i --save axios

# yarn
yarn add axios

      isSearching: false // 控制模态框显示或隐藏
    }
  },
  methods: {
    onSearch() {
      if (this.keyWord.trim() === "") {
        return alert('输入不能为空或者纯空格')
      }
      
      // 显示模态框
      this.isSearching = true
      
      this.$axios.get('http://localhost:8080/demo/article-search', {
        params: {word: this.keyWord}
      }).then(res => {
        this.searchResultList = res.data.list
        this.tips.word = this.keyWord
        this.tips.duration = res.data.duration
        this.tips.size = res.data.size
        
        // 隐藏模态框
        this.isSearching = false
        
      }).catch(() => {
        
        // 隐藏模态框
        this.isSearching = false
        
      })
    },
  }
}
</script>
