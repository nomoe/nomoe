# 2、实现首页与后端交互

# 1、测试前后端通信

首先我们要引入一个库，用于发送 ajax 请求，叫做 axios，在 github 上足足有 86k 颗 star，那为什么不引入jquery，有的人会说哥 jquery 我比较熟悉，但是哥告诉你在 vue 中引入 jquery 并不是一个合适的选择，第一，jquery 是直接操作 dom元素的，而 vue 为了提高性能则不提倡直接操作 dom；第二，jquery 这个库非常大（一万多行代码），就为了发个请求而引入这么大的库，确实有点不合理，所以我推荐使用 axios（不到两千行代码），vue 官方也推荐使用

那么，首先就要下载它，当然，这里我已经提供好了，在资料里，直接拿来用~

在 index.html 和 news-view.html中引入即可

然后在 index.js 开始编写代码

我们在 vue 实例中增加一个 created 钩子，用于在页面创建完毕就发送网络请求，当然，发送请求的逻辑我们写在一个方法里面就行，然后在然后 created 里面调用即可

# 2、模态框

之前在 jquery 版中开发了一个模态框，我们稍加改造直接拿来用即可

由于没有引入jquery，所以得修改 jk-modal.js，其实改两行就行：

function Modal(config) {
  this.width = config.width
  this.height = config.height
  this.title = config.title
  this.text = config.text
  this.init = function () {
    let template = '<div class="jk-modal-box" style="width={width}; height={height}"><h3 class="jk-modal-title">{title}</h3><p class="jk-modal-text">{text}</p></div>'
    template = template.replace("{width}", this.width + 'px')
      .replace("{height}", this.height + 'px')
      .replace("{title}", this.title)
      .replace("{text}", this.text)
    let div = document.createElement('div')
    div.id = "jk-modal"
    div.className = "jk-hide"
    div.innerHTML = template
    document.body.appendChild(div)
  }
  this.show = function () {
    // 这里改了
    document.getElementById("jk-modal").className = "jk-show"
  }
  this.hide = function () {
    // 这里改了
    document.getElementById("jk-modal").className = "jk-hide"
  }
}

# 2、开发新闻详情

首先从 jquery 版中复制获取路径参数的方法

function getQueryVariable(name) {
  let query = window.location.search.substring(1);
  let params = query.split("&");
  for (let i = 0; i < params.length; i++) {
    let pair = params[i].split("=");
    if (pair[0] === name) {
      return pair[1];
    }
  }
  return false;
}

methods: {
  getQueryVariable(name) {
    let query = window.location.search.substring(1);
    let params = query.split("&");
    for (let i = 0; i < params.length; i++) {
      let pair = params[i].split("=");

        return pair[1];

    return false;
  },
  getNewsDetail() {
    axios.get('http://localhost:8080/demo/article-view', {
      params: {id: this.getQueryVariable('id') || 1 // 没有传id则默认为1
    }
  }).
    then(res => {
      this.news = res.data
    })
  },
}
